## Tasks Overview

### Task 1 - Image Tagging with CNN (Beginner)
Built a Convolutional Neural Network from scratch using TensorFlow and Keras to classify images into 10 categories from the CIFAR-10 dataset. Achieved **74.07% test accuracy** over 25 training epochs on a Tesla T4 GPU.

**Tech Stack:** Python, TensorFlow, Keras, NumPy, Matplotlib, scikit-learn

---

### Task 2 - Store Sales and Profit Analysis (Intermediate)
Analyzed a retail superstore dataset of 8,399 orders using Python and Plotly to uncover sales trends, top-performing product categories, and customer segment profitability to support strategic business decisions.

**Tech Stack:** Python, Pandas, NumPy, Plotly

---

### Task 3 - AI-Driven NLP Project (Advanced)
Deployed Google Gemma 2 (2B-it), an open-weight instruction-tuned language model, on Google Colab with a T4 GPU. Explored its capabilities across 5 task types, visualized attention mechanisms, and compared performance against a GPT-2 baseline.

**Tech Stack:** Python, PyTorch, Hugging Face Transformers, Matplotlib
